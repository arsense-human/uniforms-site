Политика конфиденциальности

UNIFORMS использует данные из форм сайта только для обработки запроса, подготовки предварительного расчета и связи по проекту.

Какие данные могут быть переданы

Имя, контакт, комментарий, состав заказа в калькуляторе, выбранные изделия, тиражи, материалы, услуги и упаковка.

Для чего используются данные

Чтобы ответить на запрос, уточнить параметры проекта, подготовить просчет, согласовать этапы производства и передать информацию команде UNIFORMS, которая работает над заказом.

Передача данных

Данные не публикуются на сайте. При необходимости они могут быть переданы подрядчикам и производственным участникам только в объеме, который нужен для исполнения проекта.

Связь по вопросам данных

Напишите на zakaz@uniforms.ru, если нужно уточнить, изменить или удалить данные, отправленные через сайт.
